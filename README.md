# Employee Record Management System

## Overview
Employee Record Management System is a Java console-based application that allows users to manage employee records efficiently using the Java Collections Framework.

## Features
- Add Employee Records
- View All Employees
- Search Employee by ID
- Update Employee Details
- Delete Employee Records
- Fast Data Retrieval using HashMap

## Technologies Used
- Java
- OOP (Object-Oriented Programming)
- Collections Framework (HashMap)
- Console-Based User Interface

## Project Structure

```
EmployeeRecordManagement/
│
├── Employee.java
├── EmployeeManager.java
└── Main.java
```

## How to Run

### Compile

```bash
javac *.java
```

### Run

```bash
java Main
```
